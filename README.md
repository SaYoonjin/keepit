# KEEPIT

📌 개요

KEEPIT은 사용자가 주변에서 가장 가까운 짐 보관 지점을 쉽고 빠르게 찾고, 등록 및 보관할 수 있는 위치 기반 짐 보관 중개 플랫폼입니다.
여행자·외근자·통학생 등 짐을 들고 이동해야 하는 사람들의 불편을 해결하기 위해 제작되었습니다.
Appintoss(토스 개발도구) 및 DEUS 디자인 시스템을 활용해 직관적이고 실제 서비스에 가까운 프로토타입을 구현했습니다.

-------------------------------------------------------------------------------------------------------------------

🎯 목적

짐 보관소 위치·가격·이용현황 정보를 한눈에 확인할 수 있는 통합 플랫폼 제공

사용자의 거리·가격·혼잡도·선호 요소를 분석하는 AI 맞춤 추천 기능 제

사용자에게 가장 가까운 보관소를 추천하여 이동 시간 및 스트레스를 최소화

개인/소상공인이 자신의 공간을 보관소로 등록할 수 있도록 하여 수익 창출 기회 확대

여행자, 외근자 등 다양한 이용자의 편의성 향상

-------------------------------------------------------------------------------------------------------------------

🔑 핵심 기능

1️⃣ 지도 기반 보관소 탐색

현재 위치 기반 + 사용자가 설정한 우선순위에 가중치를 주어 가까운 보관소 자동 추천

거리·가격·평점 기준 필터링

지도에서 보관소 상세 정보(운영 시간, 가격, 후기 등) 확인

2️⃣ 보관소 상세 페이지

보관 가능 시간, 크기별 가격, 사진, 후기 제공

'상세페이지' 버튼 클릭 시 상세 페이지로 이동

3️⃣ 예약 및 결제 프로세스

보관할 짐 종류/규격 선택

시작/종료 시간 설정 및 30분 전, 10분 전 알림 

실시간 가격 계산

4️⃣ 보관소 등록 기능(사업자/개인)

주소 입력 + 지도 위치 선택

가격 정책, 운영 시간 입력

사용자 등록 보관소는 고객 평가를 통해 일정 수준 이하의 만족도면 삭제 처리리

5️⃣ 챗봇

"짐 보관 가능 지점 안내"

"운영 시간 문의"

🛠 적용 기술 (Tech Stack)
📱 프론트엔드 (React Native)

iOS/Android 모두 지원하는 크로스 플랫폼 앱 개발
— 하나의 코드베이스로 두 플랫폼 동시 빌드

🔧 백엔드 & DB (Spring Boot / MySQL)

사용자 인증, 보관소 등록·관리, 후기·예약 처리 등
핵심 비즈니스 로직 수행 + 데이터 저장/조회

🤖 AI 추천 시스템 (Python ML + Spring 연동)

사용자의 거리·가격·혼잡도·선호도 분석
→ 맞춤형 보관소 추천 점수 계산 및 랭킹 제공

🗺 지도 서비스 (Google Maps SDK & Places API)

지도 시각화, 현재 위치 기반 주변 보관소 탐색
장소 검색, 주소 자동완성 등 위치 기반 기능 제공

🌟 기대 효과

✔ 사용자 측면

짐 때문에 이동이 불편한 상황을 해소 → 여행 동선 효율↑

가격·거리 비교를 통해 최적의 보관소 선택 가능

앱 내에서 검색→예약→결제까지 한 번에 해결

각 지점 정보를 미리 확인하여 대기/혼잡 최소화

✔ 보관소 제공자 측면

개인 혹은 소상공인은 유휴 공간을 활용해 추가 수익 창출

KEEPIT 플랫폼을 통한 노출 증가 및 신규 고객 확보

✔ 서비스 측면

기존 코인락커 중심 시장의 문제점(위치 적음, 가격 불명확)을 해결

지역 기반 데이터 축적 → 추후 수요 예측, 맞춤 추천 서비스 확장 가능
